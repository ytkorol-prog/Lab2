Тема: **Розробка структурованих програм з розгалуженнями та повтореннями**

## Структура
- `m.py` — модуль із функціями `task_if2()`, `task_geom()`, `task_series()`
- `main.py` — меню для запуску завдань 1–3

## Як запустити
1) Встановіть Python 3.10+  
2) У цій папці виконайте:
```bash
python main.py
```

## Підказка
Якщо викладач просить «модуль + скрипт», то це саме ця структура:
- окремо логіка (функції) у `m.py`
- окремо інтерфейс запуску (меню) у `main.py`
